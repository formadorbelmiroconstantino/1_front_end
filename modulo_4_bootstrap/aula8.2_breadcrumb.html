Componentes no Bootstrap - Breadcrumb

O Breadcrumb (ou "migalha de pão") é um componente de navegação que indica a localização atual do usuário dentro da hierarquia de um site. Ele funciona como uma trilha de links que permite visualizar o caminho percorrido desde a página inicial até a página atual.

Características e Funções

  • Hierarquia Visual: Geralmente aparece no topo da página, abaixo do cabeçalho, mostrando a estrutura de pastas ou categorias (ex: Home > Eletrônicos > Smartphones).
  • Navegação Facilitada: Permite que o usuário retorne rapidamente a níveis superiores com apenas um clique.
  • Benefício para SEO: Ajuda motores de busca, como o Google, a entenderem a organização e a hierarquia do seu site, o que pode melhorar o ranqueamento.

Implementação no Bootstrap

No Bootstrap (incluindo a versão v5.3), o breadcrumb é construído usando uma lista ordenada (<ol>) dentro de um elemento <nav>. Exemplo de Código:

Separadores

Os separadores são adicionados automaticamente em CSS através de ::beforee e content. Eles podem ser alterados modificando uma propriedade personalizada local do CSS --bs-breadcrumb-divider ou através da $breadcrumb-divider variável Sass — e $breadcrumb-divider-flipped para sua contraparte RTL, se necessário. Por padrão, usamos nossa variável Sass, que é definida como alternativa à propriedade personalizada. Dessa forma, você obtém um separador global que pode ser sobrescrito sem precisar recompilar o CSS.

Melhores Práticas

  • Acessibilidade: É essencial usar o atributo aria-label="breadcrumb" no elemento <nav> e aria-current="page" no item da página atual para que leitores de tela identifiquem corretamente o componente.
  • Consistência: Utilize o breadcrumb de forma padronizada em todas as páginas internas para não confundir o usuário.
  • Simplicidade: O design deve ser discreto para não competir com o conteúdo principal da página.
