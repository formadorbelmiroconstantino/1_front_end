Formulários no Bootstrap

Os formulários do Bootstrap são componentes de interface desenhados para capturar dados de usuários em sites e aplicações web de forma organizada e visualmente atraente.

Eles oferecem as seguintes características principais:

  • Estilização Pronta: Transformam elementos padrão do HTML (como <input>, <select> e <textarea>) em campos modernos com bordas suaves, efeitos de foco e espaçamentos padronizados.
  • Responsividade: Adaptam-se automaticamente a diferentes tamanhos de tela (telemóveis, tablets e desktops) utilizando o sistema de grid do framework.
  • Controlos Diversos: Incluem estilos específicos para caixas de selecção (checkboxes), botões de rádio (radio buttons), interruptores (switches) e campos de upload de arquivos.
  • Validação: Oferecem classes para fornecer feedback visual imediato ao usuário, indicando se os dados inseridos são válidos ou inválidos através de cores (verde para sucesso, vermelho para erro) e mensagens de suporte.

Para utilizá-los, geralmente aplica-se a classe .form-control aos elementos de entrada de texto e a classe .form-label aos rótulos, garantindo o alinhamento e a estética padrão do Bootstrap.

Visão geral - Fonte: Site Oficial do Bootstrap

Os controllers de formulário do Bootstrap expandem nossos estilos de formulário Rebooted com classes. Use essas classes para optar por exibições personalizadas e obter uma renderização mais consistente em diferentes navegadores e dispositivos.

Certifique-se de usar um atributo type apropriado em todos os campos de entrada (por exemplo, email para endereço de email ou number para informações numéricas) para aproveitar os controllres de entrada mais recentes, como verificação de email, selecção de números e muito mais.

Abaixo, está um exemplo rápido para demonstrar os estilos de formulário do Bootstrap. Continue lendo para obter mais informações sobre classes obrigatórias, layout de formulário e muito mais.

<form>
<div class="mb-3">
<label for="exampleInputEmail1" class="form-label">
Email address</label>
<input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp">
<div id="emailHelp" class="form-text">We'll never share your email with anyone else.</div>
</div>
<div class="mb-3">
<label for="exampleInputPassword1" class="form-label">Password</label>
<input type="password" class="form-control" id="exampleInputPassword1">
</div>
<div class="mb-3 form-check">
<input type="checkbox" class="form-check-input" id="exampleCheck1">
<label class="form-check-label" for="exampleCheck1">Check me out</label>
</div>
<button type="submit" class="btn btn-primary">Submit</button>
</form>

Explicação

Este código utiliza o framework Bootstrap 5 (versão actual em 2026) para criar um formulário estruturado e estilizado. Aqui está a explicação detalhada de cada parte:

1. Estrutura de Agrupamento (mb-3)

O formulário é dividido em blocos usando <div class="mb-3">.

  • mb-3: É uma classe de utilitário que significa margin-bottom: 3. Ela cria um espaçamento vertical entre os campos para que eles não fiquem colados uns nos outros.

2. Campo de Email

  • <label>: O atributo for="exampleInputEmail1" conecta o rótulo ao id do input, melhorando a acessibilidade (se clicar no texto, o cursor vai para o campo). A classe .form-label aplica a fonte e o espaçamento correctos.
  • <input type="email">: Define que o campo espera um e-mail
  • class="form-control": Esta é a classe principal do Bootstrap. Ela faz o campo ocupar 100% da largura disponível, adiciona bordas arredondadas e o efeito azul ao clicar (foco).
  • aria-describedby="emailHelp": Indica que o texto explicativo abaixo (o div com id emailHelp) descreve este campo para leitores de tela.
  • .form-text: Estiliza o texto de ajuda com uma cor mais clara e fonte menor.

3. Campo de Senha

  • type="password": Garante que os caracteres digitados sejam ocultados (geralmente por asteriscos ou pontos).
  • Utiliza as mesmas classes .form-label e .form-control para manter a consistência visual.

4. Checkbox (Caixa de Seleção)

  • .form-check: Uma classe de container específica para alinhar correctamente o quadradinho de selecção e o texto ao lado.
  • .form-check-input: Estiliza o quadrado do checkbox.
  • .form-check-label: Garante o alinhamento perfeito do texto ao lado do checkbox.

5. Botão de Envio

  • type="submit": Define que este botão envia os dados do formulário.
  • .btn: Classe base para todos os botões do Bootstrap.
  • .btn-primary: Aplica a cor de destaque principal (geralmente azul), que identifica a acção principal da página.

Resumo das Classes Principais

# Classe Função
1 .form-control Estiliza inputs de texto, email e senha.
2 .form-label Estiliza os rótulos (nomes) dos campos.
3 .mb-3 Adiciona margem inferior para separar os elementos.
4 .btn .btn-primary Transforma um botão simples em um botão azul estilizado.
