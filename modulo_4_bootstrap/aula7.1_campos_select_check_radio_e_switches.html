Formulários no Bootstrap - Campos Select, Checkboxes, Radios e Switches

Para criar campos de formulário no Bootstrap (versão 5), você utiliza classes específicas para garantir que os elementos sejam responsivos e visualmente consistentes.

1. Select (Menu de Selecção)

O Bootstrap utiliza a classe .form-select para estilizar o elemento <select>.

<label for="selectExemplo" class="form-label">Escolha uma opção</label>
<select class="form-select form-select-lg mb-3" id="selectExemplo">
<option selected>Seleccione a sua província</option>
<option value="1">Kwan-Norte</option>
<option value="2">Bié</option>
</select>

Resultado:

1.1 Múltiplos Selects

Pode criar campos de múltipla selecção adicione o atributo multiple na tag <select>:

<select class="form-select" multiple aria-label="Multiple select example">
<option selected>Open this select menu</option>
<option value="1">One</option>
<option value="2">Two</option>
<option value="3">Three</option>
</select>

Resultado:



1.2 Tamanhos

Você também pode escolher entre selectores personalizados pequenos e grandes para corresponder aos tamanhos dos campos de texto de tamanho semelhante. Portanto, use .form-select-lg ou .form-select-sm para variações de tamanho.

<select class="form-select form-select-lg mb-3" aria-label="Large select example">
<option selected>Open this select menu (Menu com Tamanho Maior)</option>
<option value="1">One</option>
<option value="2">Two</option>
<option value="3">Three</option>
</select>
<select class="form-select form-select-sm" aria-label="Small select example">
<option selected>Open this select menu (Menu com Tamanho Menor)</option>
<option value="1">One</option>
<option value="2">Two</option>
<option value="3">Three</option>
</select>

Resultado:



1.3 Desabilitar o Campo Select

Adicione o atributo disabled a um elemento select para torná-lo acinzentado e remover os eventos do cursor.

<select class="form-select" aria-label="Disabled select example" disabled>
<option value="1">One</option>
<option value="2">Two</option>
<option value="3">Three</option>
</select>

Resultado:

Muito utilizado quando não quisermos que o utilizador edite o valor.

Observação: esse atributo pode ser utilizado em qualquer campo de um formulário.

2. Checkboxes (Caixas de Selecção)

Os checkboxes e rádios devem ser envolvidos em uma <div> com a classe .form-check.

<div class="form-check">
<input class="form-check-input" type="checkbox" value="" id="checkPadrao">
<label class="form-check-label" for="checkPadrao">
Checkbox 1 </label>
</div>
<div class="form-check">
<input class="form-check-input" type="checkbox" value="" id="checkDisabled">
<label class="form-check-label" for="checkDisabled">
Checkbox 2 </label>
</div>

Resultado:

3. Radios (Botões de Opção)

Funcionam de forma idêntica aos checkboxes, mas com type="radio". Lembre-se de usar o mesmo name para agrupar as opções.

<div class="form-check">
<input class="form-check-input" type="radio" name="flexRadioDefault" id="radio1" checked>
<label class="form-check-label" for="radio1">
Opção 1 </label>
</div> <div class="form-check">
<input class="form-check-input" type="radio" name="flexRadioDefault" id="radio2">
<label class="form-check-label" for="radio2">
Opção 2 </label>
</div>

Resultado:

4. Switches (Interruptores)

O Bootstrap 5 permite transformar um checkbox em um interruptor visual (toggle) usando a classe .form-switch.

<div class="form-check form-switch">
<input class="form-check-input" type="checkbox" role="switch" id="switchCheckDefault">
<label class="form-check-label" for="switchCheckDefault">Default switch checkbox input</label>
</div>
<div class="form-check form-switch">
<input class="form-check-input" type="checkbox" role="switch" id="switchCheckChecked" checked>
<label class="form-check-label" for="switchCheckChecked">Checked switch checkbox input</label>
</div>
<div class="form-check form-switch">
<input class="form-check-input" type="checkbox" role="switch" id="switchCheckDisabled" disabled>
<label class="form-check-label" for="switchCheckDisabled">Disabled switch checkbox input</label>
</div>
<div class="form-check form-switch">
<input class="form-check-input" type="checkbox" role="switch" id="switchCheckCheckedDisabled" checked disabled>
<label class="form-check-label" for="switchCheckCheckedDisabled">Disabled checked switch checkbox input</label>
</div>

Resultado:

Observação:Muito utilizado para activar ou desactivar.

5. Alinhamento Inline (Na Mesma Linha)

Para exibir checkboxes ou rádios na mesma linha, adicione a classe .form-check-inline à div principal.

5.1 Rádios na Mesma Linha

<div class="form-check form-check-inline">
<input class="form-check-input" type="radio" name="flexRadioDefault" id="radio1" checked>
<label class="form-check-label" for="radio1">
Opção 1 </label>
</div>
<div class="form-check form-check-inline">
<input class="form-check-input" type="radio" name="flexRadioDefault" id="radio2">
<label class="form-check-label" for="radio2">
Opção 2 </label>
</div>

Resultado:



5.2 Check na Mesma Linha

<div class="form-check form-check-inline">
<input class="form-check-input" type="checkbox" id="inline1" value="1">
<label class="form-check-label" for="inline1">1</label>
</div>
<div class="form-check form-check-inline">
<input class="form-check-input" type="checkbox" id="inline2" value="2"> <label class="form-check-label" for="inline2">2</label>
</div>


Resultado:



Dica de Acessibilidade: Sempre utilize o atributo id no input e o for no label correspondente, o id e o for devem ter o mesmo valor, para garantir que o campo seja clicável através do texto.

Para mais detalhes e componentes customizados, consulte a Documentação Oficial do Bootstrap.
