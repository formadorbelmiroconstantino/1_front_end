Componentes no Bootstrap - Carousel (Slide)

No Bootstrap, o Carousel (ou carrossel) é um componente de apresentação que funciona como um slider de slides para ciclar elementos como imagens ou textos.

Principais Características

As principais características deste componente incluem:

  • Multimídia: Permite exibir sequencialmente diversos tipos de conteúdo, como imagens com legendas ou banners rotativos, em um único espaço na tela
  • Controlo de Navegação: Inclui botões de "anterior" e "próximo", além de indicadores (pontos ou traços) que permitem ao utilizador saltar para slides específicos.
  • Responsividade: É construído com HTML, CSS e JavaScript, adaptando-se automaticamente a diferentes tamanhos de tela (mobile e desktop).
  • Personalização: Suporta transições suaves, reprodução automática (autoplay) e pode ser configurado para pausar ao passar o mouse.

Para implementá-lo, utiliza-se a classe principal .carousel juntamente com elementos internos como .carousel-inner (para os itens) e .carousel-item (para cada slide individual).
