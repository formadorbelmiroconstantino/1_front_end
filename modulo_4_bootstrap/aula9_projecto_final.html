Projecto Final - Site para Cursos Online

1. Preparação do Ambiente

1.1 Criação das Pastas

Crie uma pasta com o nome aula10_projecto_final_portal_ead. Dentro da pasta aula10_projecto_final_portal_ead, crie as subpastas css, img e js.

Abra essa pasta com o VSCode, dentro da subpasta css, crie o arquivo style.css e dentro da pasta js, crie o arquivo script.js.

Agora, na pasta raiz,aula10_projecto_final_portal_ead, crie o arquivo index.html e adicione a estruta básica do HTML:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>ITFocursos</title>
</head>
<body>
</body>
</html>

1.2 Carregando o CSS, os Ícones e o JS do Bootstrap

Abaixo, vamos carregar o CSS, os ícones e o JS para o Bootstrap.

1.2.1 Carregando o CSS do Bootstrap

Abaixo da tag </Title>, adicione os links do site oficial do Bootstrap: <title>ITFocursos</title>

<!--CSS Bootstrap-->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css">

1.2.2 Carregando os Ícones do Bootstrap

No site do Bootstrap, clique em Icons, copie o CDN que está mais abaixo da página.

Abaixo do carregamento do CSS do Bootstrap, cole o link dos ícones:

<!--Icons Bootstrap-->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.13.1/font/bootstrap-icons.min.css">

Por baixo do link do Bootstrap, carregue o CSS da pasta css: <link rel="stylesheet" href="css/style.css">

1.2.3 Carregando o JS do Bootstrap

Antes do fechamento da tag </body>, adicione os links Popper do site oficial do Bootstrap:
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.8/dist/umd/popper.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/js/bootstrap.min.js"></script>
