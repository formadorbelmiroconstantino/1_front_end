Componentes no Bootstrap - Badges

No Bootstrap, Badges (distintivos) são pequenos componentes de rotulagem usados para adicionar informações contextuais, contadores ou etiquetas a outros elementos de conteúdo. Eles se adaptam automaticamente ao tamanho do elemento pai (como títulos ou botões) por meio de dimensionamento relativo.

Principais usos e características:

  • Contadores de Notificação: Comumente inseridos dentro de botões ou links para mostrar números de mensagens ou alertas pendentes.
  • Destaque Visual: Utilizam cores contextuais (como bg-primary, bg-success, bg-danger) para indicar status ou categorias de forma rápida.
  • Formato Arredondado: É possível criar "pills" (formato de pílula) adicionando a classe .rounded-pill ou rounded-circle, que tornam as bordas completamente circulares.
