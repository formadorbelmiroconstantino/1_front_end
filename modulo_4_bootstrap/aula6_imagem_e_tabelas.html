Imagem e Tabelas no Bootstrap

No Bootstrap 5 (versão actual em 2026), a estilização de imagens e tabelas é feita através de classes utilitárias que garantem responsividade e design moderno.

1. Imagens

Para tornar imagens adaptáveis a diferentes tamanhos de tela e aplicar estilos rápidos:

  • Responsividade: A classe .img-fluid aplica max-width: 100% e height: auto, fazendo com que a imagem se ajuste ao elemento pai.
  • Miniaturas: Use .img-thumbnail para adicionar uma borda arredondada de 1px em volta da imagem.
  • Formas: .rounded arredonda os cantos, enquanto .rounded-circle torna a imagem circular.
  • Alinhamento: Use classes auxiliares como .float-start, .float-end ou as classes de margem automática .mx-auto .d-block para centralizar.

2. Tabelas

As tabelas no Bootstrap exigem a classe base .table para a estilização padrão.

  • Estilos Visuais:
    • .table-striped: Adiciona o efeito "zebrado" (linhas alternadas com cores diferentes).
    • .table-hover: Activa um estado de realce quando o mouse passa sobre as linhas.
    • .table-bordered: Adiciona bordas em todos os lados da tabela e células. .table-dark: Inverte as cores para um tema escuro.
  • Cores de Contexto: Você pode aplicar cores específicas a linhas (<tr>) ou células (<td>) qusando classes como .table-primary, table-success, .table-danger, entre outras..
  • Tabelas Compactas: A classe .table-sm reduz o preenchimento (padding) das células pela metade.
  • Responsividade: Envolva a tabela em um elemento com a classe .table-responsive para habilitar a barra de rolagem horizontal em dispositivos pequenos.

<div class="row">
<div class="col">

</div>>

Explicação: Essa imagem que foi adicionada ainda não obedece à responsividade. Se a tela estiver em versões pequenas, notará que a largura da imagem vai causar um scroll horizontal.

Alinhamento de Imagens

<div class="container">
<div class="row">
<div class="col">
<img src="img/responsivo.jpg" class="rounded float-start" width="200px" height="200px" alt="Imagem" title="Imagem alinha à esquerda">
<img src="img/responsivo.jpg" class="rounded float-end" width="200px" height="200px" alt="Imagem" title="Imagem alinha à direita">

</div>
</div>

Explicação

Com as classes .rounded .float-start e .rounded .float-end, alinhamos uma imagem à esquerda e a outra à direita do elemento div com classe col. .float-start para o início, esquerda, e .float-end para o final, direita. E para podermos notar bem o alinhamo, definimos uma largura e altura de 200px, imagens quadradas.

Fotografia

A tag <picture> é utilizada no Bootstrap 5 (versão actual em 2026) para oferecer diferentes versões de uma mesma imagem de acordo com o tamanho da tela ou formato de arquivo (como WebP), optimizando a performance.

Para manter a fotografia responsiva dentro do Bootstrap, você deve aplicar a classe .img-fluid directamente na tag <img> que está dentro do elemento <picture>.

Exemplo

Este exemplo carrega uma fotografia diferente dependendo da largura da janela do navegador:

<div class="container">
<div class="row">
<div class="col">
<picture class="d-flex justify-content-center align-items-center">
<!-- Imagem para telas grandes (Desktop) -->
<source srcset="img/foto-computador.jpg" media="(min-width: 992px)">
<!-- Imagem para telas médias (Tablet) -->
<source srcset="img/foto-tablet.jpg" media="(min-width: 768px)">
<!-- Imagem padrão e para dispositivos móveis -->
<img src="img/foto-telemovel.jpg" class="img-fluid rounded img-thumbnail" alt="Descrição da fotografia para acessibilidade">
</picture>
</div>
</div>
</div>

Resultado:

Explicação

  1. <source>: Define as condições de mídia. O navegador lê de cima para baixo e escolhe a primeira que for verdadeira.
  2. .img-fluid: Obrigatória para que a fotografia se comporte como uma imagem do Bootstrap, ajustando-se à largura da coluna ou container.
  3. .rounded ou .img-thumbnail: Você pode adicionar as classes de estilo (como bordas arredondadas ou molduras) na tag <img>.
  4. Optimização: Você também pode usar o <picture> para servir formatos modernos:
<picture>
<source srcset="foto.webp" type="image/webp">
<img src="img/responsivo.jpg" class="img-fluid" alt="Fotografia em formato optimizado">
</picture>

Tabelas

As tabelas no Bootstrap 5 são estilizadas para serem limpas e responsivas com o uso mínimo de classes.

Para aplicar o estilo padrão, adicione a classe .table à tag <table>.

1. Estrutura Básica

Para aplicar o estilo padrão, adicione a classe .table à tag

.

# Nome Estado Responsável Data
1 Projeto 1 Em andamento Belmiro xx/xx/xxxx
2 Projeto 2 Concluído Constantino xx/xx/xxxx

2. Classes de Estilização Comuns

Você pode combinar classes para personalizar a aparência:

  • .table-striped: Adiciona listras (zebra) nas linhas.
  • .table-hover: Activa um estado de "hover" (destaque) ao passar o mouse sobre as linhas.
  • .table-bordered: Adiciona bordas em todos os lados da tabela e células.
  • .border-class: Adiciona cor na borda da tabela (Ex.: .border-info, cor azul)
    • .table-borderless: Remove todas as bordas da tabela.
  • .table-sm: Torna a tabela mais compacta cortando o preenchimento (padding) das células.

3. Cores de Contexto

Use classes como .table-primary, .table-success, .table-danger ou .table-dark tanto na tabela inteira quanto em linhas (<tr>) ou células (<td>) específicas para indicar estados.

4. Responsividade

Para tornar uma tabela responsiva (adicionar barra de rolagem horizontal em telas pequenas), envolva a tabela em um elemento <div> com a classe .table-responsive. Mesmo sem envolvê numa div com a classe .container, ela já será responsiva.

Visão geral - Fonte (Bootstrap)

Devido ao uso generalizado de <table> elementos em widgets de terceiros, como calendários e selectores de data, as tabelas do Bootstrap são opcionais. Adicione a classe base .table qualquer `<table>` <table>, e então estenda-a com nossas classes modificadoras opcionais ou estilos personalizados. Os estilos de tabela não são herdados no Bootstrap, o que significa que qualquer tabela aninhada pode ser estilizada independentemente da tabela pai.

Usando a formatação de tabela mais básica, veja como .table, tabelas baseadas em elementos <table>, ficam no Bootstrap:

# First Last Handle
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 John Doe @social
