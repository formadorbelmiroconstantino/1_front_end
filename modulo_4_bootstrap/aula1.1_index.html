Introduçã ao Bootstrap

Além de adicionarmos o CSS e o JS para o Bootstrap, também vamos adicionar o jQuery, baixando-o a do site oficial https://jquery.com/.

Portanto, entre no site, clique em Download jQuery v3.7.1, abrirá a página para baixá-lo, clique, novamente, em Download jQuery v3.7.1, copie a URL e acima do JS do Bootstrap, o que pusemos antes de </body>, abra um script e cole dentro de src: <script src="cole aqui"></script>
