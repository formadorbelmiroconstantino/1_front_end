Conceitos sobre AJAX

AJAX (Asynchronous JavaScript and XML) é um conjunto de técnicas de desenvolvimento web que permite atualizar partes de uma página sem recarregá-la inteira, usando JavaScript para se comunicar com o servidor em segundo plano. Isso torna as aplicações web mais rápidas e interativas, pois o navegador troca dados com o servidor de forma assíncrona, atualizando apenas o conteúdo necessário, como em feeds de notícias ou formulários dinâmicos, com tecnologias como HTML, CSS, JavaScript e o objeto XMLHttpRequest (ou APIs mais modernas como Fetch).

Como funciona?

  • Assíncrono: Diferente das requisições síncronas (que bloqueiam a página), o AJAX envia e recebe dados do servidor em segundo plano enquanto o usuário continua navegando ou interagindo com a página.
  • Tecnologias Envolvidas: Combina HTML/CSS (estrutura e estilo), JavaScript (lógica e manipulação) e o DOM (Document Object Model) para actualizar a interface.
  • Comunicação com o Servidor: O JavaScript usa o objecto XMLHttpRequest (ou Fetch API) para fazer requisições ao servidor, recebendo dados (geralmente em JSON ou XML) e actualizando a página sem um reload completo.

Benefícios

  • Melhor Experiência do Usuário: Páginas mais fluidas, rápidas e responsivas.
  • Menos Tráfego de Rede: Apenas os dados necessários são transferidos, não a página inteira.

Exemplos de uso

  • Actualização de feeds de redes sociais (como o Twitter).
  • Preenchimento automático de campos de busca (como o Google Suggest).
  • Avaliações de produtos com estrelinhas em e-commerce.
  • Chats online.
