Linhas e Colunas no Bootstrap

Vamos criar uma div e no CSS, atribuiremos uma largura, uma altura e um fundo vermelho. Com margin:auto, ficará centralizada.

No HTML:

<div class="tema"></div>

No CSS:

.tema{
width: 80%;
height: 600px;
background-color:red;
margin: auto;
}

Resultado:

Tivemos de informar a largura e centralizar o container (div) com margin:auto. Com os elementos de layout do Bootstrap, não será necessário nos darmos esse trabalho porque eles fazem isso por nós.

Linhas (row) e Colunas (col)

As classes row e col desempenham uma função importante para distribuir os elementos que se estivéssemos a utilizar o FlexBox. Abaixo, vamos dar um exemplo.

HTML do Bootstrap

No site do Bootstrap, vá em Layout, clique em Grid, copie o HTML do primeiro exemplo que lá está e cole no seu arquivo HTML:

<div class="container text-center">
<div class="row">
<div class="col u"> Column 1 </div>
<div class="col d"> Column 2 </div>
<div class="col t"> Column 3 </div>
</div>
</div>

CSS Adicional

No modelo que tiramos da página do Bootstrap, adicionamos em cada coluna uma classe com os valores u de um, d de dois e t de três e formatamo-las no CSS externo desse curco com as seguintes propriedade:

/*Se adicionarmos a altura so aqui, vai afectar as 3 colunas*/
.u{
background-color: red;
height: 600px;
}
.d{
background-color: green;
height: 500px;
}
/*Não adicionamos altura nessa coluna, mas ela pegou a maior altura que definimos mais acima*/
.t{
background-color: aqua;
}

Não adicionamos altura na terceira coluna, mas ela pegou a maior altura que definimos mais acima e também não foi necessário centralizarmos nem definirmos a largura total da tela porque o cantainer fez tudo isso por nós. Olhe que se quiser definir uma altura uniforme/geral para todas as colunas, basta defini-la no .row com width:600px ou com um valor que achar melhor.

Elementos Ocupando Tamanhos Diferentes no Bootstrap

Para fazer elementos ocuparem tamanhos de colunas diferentes no Bootstrap, use classes como col-quantidade de colunas, lembrando que o modelo do Bootstrap é de 12 colunas, dentro de uma div com a classe row, aproveitando , onde a soma dos números das colunas na mesma linha não pode exceder 12, e o número indica quantas das 12 divisões cada elemento ocupará.

Exemplo

<div class="container">
<div class="row">
<!-- Coluna 1: Ocupa 4 colunas -->
<div class="col-4 u">
Conteúdo da Coluna 1 (4/12) </div>
<!-- Coluna 2: Ocupa 6 colunas -->
<div class="col-6 d">
Conteúdo da Coluna 2 (6/12) </div>
<!-- Coluna 3: Ocupa 2 colunas -->
<div class="col-2 t">
Conteúdo da Coluna 3 (2/12) </div>
</div>
</div>

Resultado:

Conteúdo da Coluna 1 (4/12)
Conteúdo da Coluna 2 (6/12)
Conteúdo da Coluna 3 (2/12)

Observação importante: só poderá ter o resultado esperado se o seu arquivo HTML estiver bem configurado como por exemplo a importação dos links Bootstrap e do nosso style.css.

Como funciona:

  2. <div class="row">: Define uma linha na grid.
  3. <div class="col-X">: Define uma coluna. O número X (de 1 a 12) indica quantas das 12 colunas disponíveis ela ocupará.
  4. Soma: No exemplo, 4 + 6 + 2 = 12, preenchendo a linha inteira. Se a soma for menor que 12, haverá espaço vazio; se for maior, as colunas seguintes quebrarão para a próxima linha.

Responsividade (Opcional, mas recomendado): Você pode combinar tamanhos para diferentes telas (xs, sm, md, lg, xl, xxl) para layouts responsivos.

  • col-md-4: Ocupa 4 colunas em telas médias e maiores.
  • col-sm-6: Ocupa 6 colunas em telas pequenas e maiores.

No HTML:

<div class="row">
<div class="col-12 col-md-4 u">Coluna 1 (12 no mobile, 4 no desktop)</div>
<div class="col-12 col-md-6 d">Coluna 2 (12 no mobile, 6 no desktop)</div>
<div class="col-12 col-md-2 t">Coluna 3 (12 no mobile, 2 no desktop)</div>
</div>

No CSS:

.row{
margin-bottom: 20px;
}
.u{
background-color: red;
height: 600px;
}
.d{
background-color: green;
height: 500px;
}
.t{
background-color: aqua;
height: 700px;
}

Resultado:

Coluna 1 (12 no mobile, 4 no desktop)
Coluna 2 (12 no mobile, 6 no desktop)
Coluna 3 (12 no mobile, 2 no desktop)

Este exemplo mostra um layout flexível, ideal para design responsivo, usando as classes col-md-X e col-12, sendo que col-12 garante que em telas menores (mobile), cada um ocupe a largura total (12/12).

Distribuindo os Elementos para a Responsividade com o Uso dos Breakpoints

Para definirmos a responsividade no submódulo de HTML e CSS, tivemos de utilizar o media query. No entanto, com o auxílio de cols na row e com o Breakpoints fica mais fácil.

Para criar layouts responsivos no Bootstrap (ou frameworks similares baseados em Flexbox), utiliza-se, como já vimos, a estrutura de 12 colunas. A distribuição dos elementos col dentro de uma row é controlada por prefixos que definem o comportamento em diferentes tamanhos de tela.

Sintaxe dos Breakpoints

A estrutura segue o formato col-{breakpoint}-{número}, onde o número representa quantas das 12 colunas o elemento ocupará:

  • col-: Telas extra pequenas (telemóveis < 576px).
  • col-sm-: Telas pequenas (telemóveis em landscape ≥ 576px).
  • col-md-: Telas médias (tablets ≥ 768px).
  • col-lg-: Telas grandes (desktops ≥ 992px).
  • col-xl-: Telas extra grandes (monitores ≥ 1200px).
  • col-xxl-: Telas ultra grandes (telas ≥ 1400px).

Exemplo Prático de Distribuição

Se você deseja que três elementos fiquem empilhados no celular, mas lado a lado no desktop, utilize:

No HTML

<div class="container"> <div class="row">
<!-- Ocupa 12 colunas (100%) no celular e 4 colunas (33%) no desktop -->
<div class="col-12 col-md-4 u">Coluna 1</div>
<div class="col-12 col-md-4 d">Coluna 2</div>
<div class="col-12 col-md-4 t">Coluna 3</div>
</div>
</div>

Resultado:

Coluna 1
Coluna 2
Coluna 3

Regras de Ouro para Responsividade

  1. Soma de 12: Se a soma das classes em uma linha exceder 12, o próximo elemento quebrará automaticamente para a linha de baixo.
  2. Mobile-First: As classes sem breakpoint (ex: col-6) aplicam-se a todos os tamanhos, a menos que sejam subscritas por um breakpoint maior (ex: col-6 col-lg-3).
  3. Auto-layout: Usar apenas .col sem número faz com que o framework divida o espaço igualmente entre todos os elementos da row.

Para documentação técnica detalhada e testes de grid, consulte o Guia de Grid do Bootstrap ou explore recursos de design no MDN Web Docs sobre Flexbox.

Explicação do Comportamento (Breakpoints)

Ao redimensionar a janela, o comportamento será o seguinte:

  1. col-12 (< 576px): Cada item ocupa a largura total (100%). Você verá um item abaixo do outro.
  2. col-sm-6 (≥ 576px): Cada item ocupa metade da largura (50%). Verá 2 itens por linha.
  3. col-md-4 (≥ 768px): Cada item ocupa um terço (33.3%). Verá 3 itens por linha (o 4º item pula para baixo).
  4. col-lg-3 (≥ 992px): Cada item ocupa um quarto (25%). Todos os 4 itens ficam na mesma linha.
  5. col-xl-2 (≥ 1200px): Cada item ocupa 2 das 12 colunas (aprox. 16%). Sobrará espaço vazio na linha.
  6. col-xxl-1 (≥ 1400px): Cada item ocupa apenas 1 das 12 colunas (aprox. 8%). Ficarão bem estreitos em telas ultra-wide.
