Componentes no Bootstrap

Os componentes do Bootstrap são elementos de interface pré-construídos e reutilizáveis, fundamentais para a criação de sites responsivos e consistentes [1, 2]. Na versão mais recente (v5.3), os principais componentes incluem:

1. Navegação e Estrutura

  • Navbar: Barras de navegação responsivas que suportam branding, links e formulários.
  • Breadcrumbs: Indicadores de hierarquia de páginas.
  • Pagination: Controles para navegar entre múltiplas páginas de conteúdo.

2. Feedback e Alertas

  • Alerts: Mensagens de feedback contextual (sucesso, perigo, aviso).
  • Badges: Pequenos indicadores numéricos ou de status para rotulagem rápida.
  • Toasts: Notificações leves que aparecem no canto da tela.
  • Progress: Barras para indicar o progresso de uma tarefa.

3. Interatividade e Dinâmica

  • Modals: Janelas sobrepostas para diálogos, formulários ou confirmações.
  • Tooltips e Popovers: Pequenas janelas de ajuda que surgem ao passar o mouse ou clicar.
  • Accordion: Painéis que expandem ou colapsam para organizar grandes volumes de texto.
  • Carousel: Slideshow para apresentar imagens ou conteúdos cíclicos.

4. Conteúdo e Formulários

  • Cards: Containers flexíveis que agrupam imagens, textos e botões.
  • Buttons: Estilos padronizados para ações, incluindo grupos de botões.
  • Forms: Controles customizados como campos de texto, checkboxes e seletores.
  • List Groups: Séries de itens de conteúdo em listas flexíveis.
