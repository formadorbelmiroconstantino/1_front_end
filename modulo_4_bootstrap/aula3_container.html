Container no Bootstrap

No Bootstrap, um Container é um bloco de construção fundamental para criar layouts responsivos, sendo uma <div> ou outro elemento que envolve e centraliza o conteúdo, definindo suas margens e largura de forma adaptável a diferentes tamanhos de tela (responsividade). Existem os tipos .container (largura fixa com breakpoints), .container-fluid (largura total, 100%) e .container-{breakpoint} (100% até um ponto de quebra específico).

Tipos Principais de Containeres:

  • .container: Define uma largura máxima fixa que aumenta em "breakpoints" (tamanhos de tela específicos), mantendo o conteúdo centralizado e com margens laterais.
  • .container-fluid: Ocupa 100% da largura da tela em todos os breakpoints, ideal para fundos ou seções que precisam se estender por toda a viewport.
  • .container-{breakpoint}: Usa 100% da largura até um ponto de quebra (ex: .container-sm, .container-md, .container-lg), a partir do qual ele se torna fixo e centralizado, como o .container padrão, mas em um tamanho diferente.

Para que serve?

  • Estrutura: É a base para o sistema de grid do Bootstrap, contendo linhas (.row) e colunas (.col).
  • Responsividade: Garante que o conteúdo se ajuste bem a desktops, tablets e celulares.
  • Alinhamento: Centraliza o conteúdo na página, melhorando a legibilidade.
  • Padronização: Aplica margens internas (padding) e limites de largura de forma consistente em todo o projecto.

Exemplo de Uso

Abra o site do Bootstrap, clique em Docs, no lado esquerdo, procure a secção de Layout e clique em Container. Lê o que eles falam sobre o seu Container Padrão:

Default container

"Nossa classe .container padrão é um container responsivo de largura fixa, o que significa que a max-width se adapta a cada ponto de interrupção."

<div class="container">
<!-- Content here -->
</div>

Logo, com o uso da classe container do Bootstrap, não precisamos nos preocupar em definir a largura porque esse framework já faz isso por nós. Diferente do que tivemos de fazer no módulo de HTML e CSS, que tínhamos de definir a largula dos elementos no CSS com width:valor.
