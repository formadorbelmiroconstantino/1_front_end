Formulários no Bootstrap - Campo para Rótulos Flutuantes

Crie rótulos de formulário simples e elegantes que flutuam sobre seus campos de entrada.

Envolva um par de elementos <div> <input class="form-control"> e <span> em <div> para habilitar rótulos flutuantes com os campos de formulário de texto do Bootstrap.<label>.form-floating

placeholderÉ necessário que cada elemento contenha um atributo não vazio, <input> pois nossa implementação de rótulo flutuante, que utiliza apenas CSS, depende do :placeholder-shownpseudo-elemento para detectar quando o campo de entrada está vazio. O texto de espaço reservado em si não é visível; apenas o elemento <label> é mostrado aos usuários.



Observe também que o elemento <input> deve vir primeiro para que possamos utilizar um selector de elemento irmão (ou seja, ~).

Para definir uma altura personalizada para o seu elemento <textarea>, não use o atributo rows. Em vez disso, defina uma altura explícita height(seja inline ou via CSS personalizado).
