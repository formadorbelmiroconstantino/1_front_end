Trabalhando com Links em HTML

Trabalhar com links em HTML envolve usar a tag (âncora) com o atributo href para direccionar o utilizador/usuário para outros recursos, como páginas ou sites externos, usando URLs absolutas (https://...) ou relativas (caminhos dentro do seu site). Para links internos (na mesma página), use # seguido do id do elemento de destino. O texto ou imagem entre e é o que o utilizador clica, e o atributo target="_blank" abre o link em uma nova aba.

Sintaxe Básica

  • < a>: A tag de âncora.
  • href: Atributo que contém o destino do link (URL).
  • Texto/Imagem: Conteúdo clicável.

Boas Práticas

  • Links Internos: Use caminhos relativos para facilitar a organização do site.
  • Experiência do Usuário: Use target="_blank" com moderação para não "expulsar" o usuário do seu site.
