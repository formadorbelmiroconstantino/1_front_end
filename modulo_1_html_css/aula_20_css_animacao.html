CSS - Animações em HTML

Trabalhar com animações CSS permite criar sequências complexas de movimentos que vão além das transições simples, pois permitem múltiplos estados intermediários. Para criar uma animação, você precisa de dois componentes principais: a regra @keyframes e a propriedade animation aplicada ao elemento.

1. Definindo a Sequência com @keyframes

A regra @keyframes define o que acontece em momentos específicos da animação, usando porcentagens (0% a 100%) ou as palavras-chave from (0%) e to (100%).

CSS

@keyframes flutuar { 0% { transform: translateY(0); } 50% { transform: translateY(-20px); background-color: #3498db; } 100% { transform: translateY(0); } }

2. Aplicando a Animação ao Elemento

Após definir os frames, você deve "ligar" a animação ao elemento desejado usando as propriedades de configuração:

  • animation-name: O nome dado no @keyframes.
  • animation-duration: Duração total (ex: 2s).
  • animation-iteration-count: Quantas vezes repetir (ex: 3 ou infinite).
  • animation-timing-function: Velocidade do movimento (ex: ease, linear).
  • animation-fill-mode: Define se o elemento volta ao estado inicial ou mantém o estado final (forwards) após o término.

Exemplo de uso abreviado:

.elemento { /* nome | duração | curva | repetição */ animation: flutuar 3s ease-in-out infinite; }

3. Controle de Estado (Play/Pause)

Diferente das transições, você pode pausar uma animação em execução usando a propriedade animation-play-state.

.elemento:hover { animation-play-state: paused; /* Pausa ao passar o mouse */ }

4. Boas Práticas para 2025

  • Performance: Para animações fluidas a 60 FPS, foque em animar apenas transform e opacity. Evite animar propriedades que afetam o layout (como width, height ou margin), pois elas exigem mais processamento do navegador.
  • Acessibilidade: Utilize @media (prefers-reduced-motion: reduce) para desativar animações para usuários que sentem desconforto com movimentos intensos.
  • Bibliotecas: Em 2025, para animações extremamente complexas ou baseadas em scroll, desenvolvedores utilizam ferramentas como a Animate.css ou o GSAP para maior controle.
