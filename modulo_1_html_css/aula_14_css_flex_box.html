CSS - FlexBox

O que é FlexBox?

Flexbox (Flexible Box Layout) é uma ferramenta do CSS3 para criar layouts flexíveis e responsivos, organizando itens em uma dimensão (linha ou coluna) de forma eficiente, usando um contêiner pai (display: flex;) e seus itens filhos, permitindo alinhar, distribuir espaços e controlar o fluxo dos elementos sem precisar de floats ou posicionamentos complexos. Ele se adapta a diferentes tamanhos de tela, tornando o desenvolvimento mais fácil e dinâmico, e tem propriedades para o contêiner (pai) e para os itens (filhos), como flex-direction, justify-content, align-items, e flex-wrap

Como funciona (HTML e CSS)

No HTML: Você define um elemento como contêiner e seus filhos como itens

No CSS com o auxílio de style no head: Aplique display: flex; ao elemento pai (.container)

.container { display: flex; /* Transforma o container em um flex container */ /* Propriedades adicionais para o pai */ flex-direction: row; /* Direção padrão: linha */ justify-content: center; /* Alinha no eixo principal */ align-items: center; /* Alinha no eixo transversal */ flex-wrap: wrap; /* Permite que os itens quebrem para a próxima linha */ } .item { /* Propriedades para os filhos */ /* flex-grow, flex-shrink, flex-basis (atalho: flex) */ /* order */ }

Propriedades Chave

No Contêiner (Pai)

  • display: flex; : O principal comando para activar o Flexbox.
  • flex-direction: Define o eixo principal (row, column, row-reverse, column-reverse).
  • justify-content: Alinha itens no eixo principal (flex-start, flex-end, center, space-between, etc.).
  • align-items: Alinha itens no eixo transversal (stretch, flex-start, flex-end, center, baseline).
  • flex-wrap: Controla se os itens quebram para uma nova linha (nowrap, wrap, wrap-reverse).
  • flex-flow: Atalho para flex-direction e flex-wrap.

Nos Itens (Filhos)

  • flex-grow: Define a capacidade de um item crescer para preencher espaço extra.
  • flex-shrink: Define a capacidade de um item encolher.
  • flex-basis: Define o tamanho inicial de um item.
  • flex: Atalho para flex-grow, flex-shrink e flex-basis.
  • order: Altera a ordem visual dos itens sem mudar o código HTML, afetando a acessibilidade.

Vantagens

  • Layouts Responsivos: Adapta-se a telas de diversos tamanhos.
  • Alinhamento Fácil: Centralizar itens é simples (ex: justify-content: center; e align-items: center;).
  • Distribuição de Espaço: Controle sobre como o espaço é distribuído entre os itens.
  • Ordem Visual Flexível: Reordene elementos sem alterar o HTML (com a propriedade order).
