O que são listas por definição em HTML?

Listas por definição em HTML (tags < dl>, < dt>, < dd>) são usadas para criar glossários ou dicionários, associando termos (< dt>) às suas descrições (< dd>), dentro de um contêiner < dl> (Definition List). Elas são ideais para pares de itens onde a ordem não importa tanto quanto a relação entre o termo e sua explicação, como em catálogos de produtos ou perguntas frequentes.

Como funcionam?

  • < dl> (Definition List): É o contêiner principal que envolve toda a lista de definições.
  • < dt> (Definition Term): Define o termo ou a palavra que será definida (alinhado à esquerda).
  • < dd> (Definition Description): Fornece a descrição ou definição para o termo anterior (geralmente indentado).

Exemplo

HTML
Linguagem de Marcação de Hipertexto usada para criar páginas web.
CSS
Linguagem de Folhas de Estilo para estilizar elementos de um documento.
Java Script
JavaScript (JS) é uma linguagem de programação essencial para a web, usada para criar conteúdo dinâmico e interativo em sites, funcionando no navegador do usuário para adicionar funcionalidades como animações, mapas interactivos, validação de formulários e actualizações em tempo real, sendo uma das três tecnologias fundamentais da web junto com HTML e CSS. Ele não deve ser confundido com Java, embora compartilhem parte do nome; JS é mais focado no front-end e no comportamento do usuário, enquanto Java é mais usado no back-end.
Java
Java é uma linguagem de programação orientada a objetos e uma plataforma de computação amplamente usada, conhecida por sua portabilidade ("Escreva uma vez, execute em qualquer lugar") graças à Máquina Virtual Java (JVM), que permite rodar o mesmo código em diversos sistemas operacionais. É usado para criar desde aplicativos móveis, sistemas corporativos, até aplicações web e Big Data, sendo uma tecnologia robusta, segura e escalável para empresas e desenvolvedores em todo o mundo.
Bootstrap
Bootstrap é um framework front-end de código aberto, gratuito, com componentes pré-estilizados em HTML, CSS e JavaScript, que permite criar sites e aplicações web responsivas (que se adaptam a qualquer tela, de celular a desktop) de forma rápida e eficiente, seguindo uma abordagem mobile-first (projetado primeiro para dispositivos móveis). Ele oferece um kit de ferramentas com botões, formulários, grids, modais e mais, poupando tempo de desenvolvimento ao não precisar codificar tudo do zero, sendo uma ferramenta essencial para desenvolvedores front-end.
