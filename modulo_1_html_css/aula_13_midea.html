Mídia em HTML

O que são mídia em HTML?

Mídia em HTML refere-se à incorporação de conteúdo multimídia (imagens, áudios, vídeos) diretamente em páginas web usando tags específicas como ,

Principais tags e elementos:

  • < img>: Para incluir imagens (JPEG, PNG, SVG, etc.).
  • < audio>: Para áudios (músicas, podcasts), suportando formatos como MP3, WAV, OGG.
  • < video>: Para vídeos, com suporte a MP4, WebM, OGG.
  • < iframe>: Para incorporar conteúdo de outras páginas ou sites (como vídeos do YouTube).
  • < map> e < area>: Para criar "mapas de imagem" onde diferentes áreas da imagem são clicáveis e levam a links diferentes.

Como funciona:

  • Sem plugins: O HTML5 permitiu que navegadores reproduzam áudio e vídeo nactivamente, sem a necessidade de Flash ou outros plugins, tornando os sites mais leves e rápidos.
  • Múltiplas fontes: Para garantir compatibilidade entre navegadores, você pode fornecer várias fontes de mídia (diferentes formatos) dentro das tags
  • Atributos: As tags possuem atributos para controlar o comportamento, como src (fonte), controls (mostrar controles), autoplay (reprodução automática) e width/height (dimensões).

Em resumo, mídia em HTML é a capacidade de adicionar esses elementos interativos e visuais para enriquecer o conteúdo de uma página, contando histórias e guiando o usuário de forma mais envolvente.

3. Atributos Principais

Você pode personalizar o comportamento do player com estes atributos:

  • controls: Exibe botões de play, pause, volume e progresso.
  • autoplay: Inicia o áudio automaticamente (bloqueado por muitos navegadores modernos se não houver interação prévia do usuário).
  • loop: Faz o áudio reiniciar automaticamente ao terminar.
  • muted: Inicia o áudio sem som.
  • preload: Define como o arquivo deve carregar (auto, metadata ou none).

Vídeos em HTML

Trabalhar com vídeos em HTML envolve usar a tag < video> com o atributo controls para exibir o player, e a tag < source> para especificar o arquivo de vídeo (com src e type, como video/mp4 ou video/webm) para compatibilidade entre navegadores, permitindo autoplay e outras funções com atributos, ou usar um código de incorporação (< iframe>) de plataformas como YouTube para conteúdo externo.

1. Incorporando Vídeos Locais (Seu Próprio Arquivo)

  • < video>: A tag principal para vídeos.
    • controls: Adiciona os controles de reprodução (play, pause, volume).
    • width, height: Define as dimensões do player.
  • < source>: Especifica o arquivo de vídeo. Use múltiplos < source> para diferentes formatos (MP4, WebM, Ogg) para garantir compatibilidade.
    • src: Caminho para o arquivo de vídeo.
    • type: O tipo MIME do vídeo (ex: video/mp4).
