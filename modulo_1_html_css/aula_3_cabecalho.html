Cabeçalho

O que é?

Em HTML, "cabeçalho" pode se referir a duas coisas principais: a tag < head>, que contém metadados (informações
sobre a página, não visíveis) e a tag < h1> a < h6>, que definem títulos e subtítulos para o conteúdo visível,
organizando a hierarquia do texto. O elemento < header> (com 'er' no final, do HTML5) é usado para agrupar
conteúdo introdutório ou de navegação, como logo, menu e formulário de busca, aparecendo no topo da página visível.

Nós ignoramos o cabeçalho que começa com < head> e termina com < /head> quando estamos a falar de conteúdos porque essa tag é utilizada, como já nos referimos mais acima, para as configurações invisíveis do site
