Tabelas em HTML

O que são tabelas em HTML?

Tabelas em HTML são estruturas para organizar dados em linhas e colunas, usando tags como < table>, < tr> (linha) e < td> (célula), para exibir informações de forma estruturada, como listas ou relatórios, sendo a forma semântica correta para dados tabulares, e não para layouts de página, que devem ser estilizados com CSS. Elas permitem separar cabeçalhos (), corpo () e rodapé (), facilitando a apresentação de dados complexos na web.

Principais Tags

  • < table>: Define o início e o fim da tabela.
  • < tr> (Table Row): Define uma linha na tabela.
  • < td> (Table Data): Define uma célula de dados dentro de uma linha.
  • < th> (Table Header): Define uma célula de cabeçalho (texto em negrito e centralizado por padrão).

Estrutura semântica (opcional, mas recomendado)

  • < thead>: Agrupa as linhas de cabeçalho.
  • < tbody>: Agrupa as linhas de dados principais (corpo da tabela).
  • < tfoot>: Agrupa as linhas de rodapé (totais, resumos).

Para que servem

  • Exibir dados tabulares: Listas de produtos, estatísticas, orçamentos, cadastros de usuários, placares de jogos, etc..
  • Comparar e classificar informações: Dados que precisam ser cruzados ou organizados. O que não fazer
Não usar para layout: Antigamente, usavam-se tabelas para posicionar elementos da página, mas hoje isso é considerado uma má prática; para isso, use CSS (Flexbox ou Grid).

Células que abrangem mais que uma linha/coluna

Em algumas situações, temos a necessidade de utilizar uma mesclagem de célula, ou seja, criar uma célula que abrange mais do que uma linha ou coluna. Dessa forma, podemos utilizar os atributos colspan e rowspan. O colspan=”x” vai fazer uma mesclagem de colunas, e o rowspan=”x” vai mesclar linhas, bastando substituir o x pelo número de colunas ou linhas que deseja ocupar. Além disso, podemos também mesclar os dois em uma mesma célula. Com isso, vamos ver os exemplos a seguir:

Primeiro, utilizando o colspan:

Nome Telefones
Belmiro Constantino 8888-8888 9999-9999
Junior 1111-1111 2222-2222

Depois, utilizando o rowspan:

Primeiro Nome: Belmiro Constantino
Telefone: 8888-8888
9999-9999

Utilizando o colgroup e col

Anteriormente, explicamos que hoje em dia, boa parte dos atributos de uma tabela se encontra depreciado. Portanto, a estilização deve ser feita através do CSS. Dessa forma, existe um elemento muito importante que nos permite criar um grupo de colunas, podendo assim estilizar diversas colunas de uma só vez. Esse elemento é a tag < colgroup>. Posteriormente, dentro do grupo de colunas, incluímos então as tags < col/> para poder definir cada coluna.

Vejamos então o seguinte exemplo, aplicando o CSS às classes que utilizaremos nas < col>:

Nome Idade Profissao
Belmiro Constantino 22 Director da ITFocursos
Junior 26 Escritor

Além dessa forma de trabalhar, podemos ainda aplicar um grupo que abranja mais que uma coluna, utilizando o atributo span=”x”. Assim, basta substituir o x pelo número de colunas que desejamos atribuir esse grupo.

Em seguida, vejamos então o exemplo:

Nome Idade Profissao
