CSS - FlexBox | flex-wrap

Trabalhar com flex-wrap no HTML (via CSS) é essencial para layouts responsivos, permitindo que itens Flexbox quebrem para a próxima linha quando não há espaço, em vez de ficarem todos em uma única linha e menores; basta aplicar display: flex; ao container e flex-wrap: wrap; para quebrar a linha e flex-wrap: wrap-reverse; para inverter a ordem, controlando o comportamento dos filhos em diferentes tamanhos de tela.

Como usar flex-wrap

  1. Defina um Flex Container: No seu HTML, tenha um elemento pai (container) e elementos filhos (flex items).
  2. Aplique display: flex;: No CSS, no seletor do container, adicione display: flex; para ativar o Flexbox.
  3. Adicione flex-wrap:
    1. flex-wrap: wrap; (Mais comum): Faz com que os itens quebrem para uma nova linha quando não couberem na largura do container, mantendo o tamanho dos itens (ótimo para responsividade).
    2. flex-wrap: nowrap; (Padrão): Os itens tentam ficar na mesma linha, encolhendo se necessário, e não quebram.
    3. flex-wrap: wrap-reverse;: Quebra os itens para a linha de cima, invertendo a ordem das linhas.

Benefícios para Design Responsivo

flex-wrap é uma ferramenta poderosa para criar layouts que se adaptam a diferentes telas (celular, tablet, desktop), pois ele garante que os elementos se reorganizem naturalmente, sem precisar de muitos media queries complexos, como visto em tutoriais e documentações como MDN Web Docs e Origamid.
