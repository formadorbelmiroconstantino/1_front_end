O que são atributos em html?

Atributos HTML são palavras especiais dentro da tag de abertura que fornecem informações extras ou controlam o comportamento de um elemento, agindo como modificadores para personalizar sua função ou aparência, como definir um link (href), uma imagem (src), uma classe (class), ou um ID único (id), sempre no formato nome="valor". Eles são essenciais para tornar o conteúdo dinâmico, identificável e estilizado, existindo atributos globais (para quase todas as tags) e específicos (para tags em particular).

Como funcionam

Sintaxe: São adicionados na tag de abertura, como nome="valor" (ex: <img src="imagem.jpg">).
Função: Adicionam dados (metadados) ou modificam a funcionalidade da tag.

Exemplos de Atributos Comuns

href: Define o destino de um link (<a>) ou recurso;

src: Especifica o caminho de um arquivo de mídia, como uma imagem (<img>) ou vídeo;

alt: Fornece um texto alternativo para imagens (importante para acessibilidade e SEO);

class: Atribui uma classe CSS para estilizar um ou mais elementos;

id: Atribui um identificador único a um elemento, para estilos ou scripts específicos;

style: Permite aplicar estilos CSS diretamente ao elemento;

name: Usado em formulários para identificar campos (como em < input>);

charset: Um atributo que especifica a codificação de caracteres (conjunto de caracteres).

Tipos de Atributos

Globais: Funcionam em quase todas as tags HTML (ex: class, id, style, lang).

Específicos: Funcionam apenas em determinadas tags (ex: src e alt em ).

A tag < meta> com os dois atributos name e content, <meta name=" viewport" content=" width=device-width, initial-scale=1">, em HTML é essencial para o design responsivo, dizendo ao navegador para ajustar a largura da página à largura da tela do dispositivo (width=device-width) e definir o nível de zoom inicial como 100% (initial-scale=1), garantindo que sites fiquem legíveis e bem dimensionados em celulares e tablets, sem precisar de zoom manual.

O que cada parte faz:

.meta name="viewport": Informa ao navegador que estamos configurando a "área visível" (viewport) da página.

.width=device-width: Define a largura da página para ser igual à largura da tela do dispositivo (em pixels), fazendo com que o layout se adapte.

.initial-scale=1: Define o nível de zoom inicial quando a página é carregada, ou seja, 1:1 (sem zoom), para que um pixel do CSS seja equivalente a um pixel do dispositivo.

Por que é importante?

Sem essa tag, em dispositivos móveis, o navegador tenta renderizar a página como se fosse um desktop (com largura fixa, como 980px), e depois a reduz para caber na tela, deixando tudo muito pequeno e difícil de ler. Essa tag permite que seu site se adapte automaticamente a qualquer tamanho de tela, essencial para uma boa experiência do usuário (UX) e para o SEO.

Em resumo, atributos são as "características" ou "informações adicionais" que você dá às suas tags HTML para que elas funcionem exatamente como você precisa na página.
