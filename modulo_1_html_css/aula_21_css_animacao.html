CSS - Animações em HTML

Daremos um exemplo clássico e moderno de uma animação CSS: um Spinner de Carregamento (Loader). Este tipo de animação é muito comum em sites para indicar que um conteúdo está sendo processado.

Exemplo: Spinner de Carregamento

Este exemplo usa @keyframes para girar continuamente um círculo com uma borda colorida.


Explicação do Código:

  1. border-radius: 50%: Transforma uma div quadrada em um círculo perfeito.
  2. border-top: Ao colorir apenas um lado da borda, criamos o efeito visual de uma "faixa" que se move.
  3. @keyframes girar: Define que o elemento deve começar em 0 graus e terminar em 360 graus.
  4. animation:
    1. girar: O nome da animação que criamos.
    2. 1s: A velocidade (um giro por segundo).
    3. linear: Mantém a velocidade constante (sem acelerar ou desacelerar).
    4. infinite: Garante que ela nunca pare.

Para explorar mais estilos, você pode consultar bibliotecas como o Animate.css ou ver exemplos interativos no MDN Web Docs.
