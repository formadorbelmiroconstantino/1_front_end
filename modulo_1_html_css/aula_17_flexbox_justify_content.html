CSS FlexBox | Justify Content

Para trabalhar com justify-content no HTML usando CSS Flexbox, você define display: flex no contêiner e usa justify-content com valores como flex-start, center, flex-end, space-between, space-around ou space-evenly para alinhar os itens filhos ao longo do eixo principal (horizontal ou vertical, dependendo do flex-direction), controlando o espaço livre e a posição dos elementos.

Como usar justify-content

1. No HTML: Crie um contêiner e seus itens flexíveis.

2. No CSS (no contêiner): Aplique display: flex

.container { display: flex; /* flex-direction: row; (padrão) ou column */ }

3. Use justify-content para alinhar:

  1. flex-start: Alinha os itens no início do eixo principal (esquerda, se row).
  2. flex-end: Alinha os itens no final do eixo principal (direita, se row).
  3. center: Centraliza os itens no eixo principal.
  4. space-between: Espaça os itens igualmente; o primeiro e o último ficam nas extremidades.
  5. space-around: Espaço igual em todas as direções, com o dobro do espaço entre os itens do que nas bordas.
  6. space-evenly: Espaço igual entre todos os itens e nas extremidades.
.container { display: flex; justify-content: center; /* Exemplo: centraliza os itens */ /* justify-content: space-between; */ /* justify-content: flex-end; */ }

Observações importantes

  • justify-content sempre actua no eixo principal (main axis), determinado por flex-direction (row ou column).
  • Para alinhar no eixo secundário (perpendicular), usa-se align-items.
