O que são listas em HTML?

Listas em HTML são estruturas para organizar itens em conjuntos, podendo ser não ordenadas (< ul>) com marcadores (bolinhas), ordenadas (< ol>) com números/letras/romanos (indicando sequência), ou de definição (< dl>) para termos e suas descrições, utilizando tags como < li> (item), < dt> (termo) e < dd> (descrição) para estruturar o conteúdo, sendo essenciais para menus de navegação, instruções ou glossários.

Tipos de Listas em HTML

Lista Não Ordenada (< ul>):

Usada para itens sem ordem específica (ex: lista de compras). Marcadores padrão são círculos (bolinhas), mas podem ser estilizados. Exemplo:
  • Maçã
  • Banana
.

Exemplo de Lista Não Ordenada (< ul>):

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Lista Ordenada (< ol>):

  1. Para itens em sequência (ex: instruções passo a passo).
  2. Padrão: números (1, 2, 3), mas pode usar letras ou romanos com CSS.
  3. Exemplo:
    1. Primeiro passo
    2. Segundo passo
      3. .

    Exemplo de Lista Ordenada (< ol>):

    1. Item 1
    2. Item 2
    3. Item 3

    Lista de Definição (< dl>):

    Para apresentar termos e suas descrições (como um dicionário). Usa < dt> para o termo e < dd> para a descrição. Exemplo:
    Café
    Bebida escura
    .

    Tags Comuns

    • < ul>: Define uma lista não ordenada
    • < ol>: Define uma lista ordenada.
    • < li>: Define um item dentro de < ul> ou < ol>.
    • < dl>: Define uma lista de definição.
    • < dt>: Define um termo em uma lista de definição.
    • < dd>: Define a descrição de um termo em uma lista de definição.
