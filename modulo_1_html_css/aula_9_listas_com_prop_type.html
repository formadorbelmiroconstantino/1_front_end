Listas em HTML com a Propriedade type

A propriedade type em listas ordenadas (< ol>) no HTML serve para definir o estilo da numeração ou marcador dos itens, como números (1, 2, 3), letras (a, b, c), ou algarismos romanos (i, ii, iii), embora hoje seja mais comum usar CSS (com a propriedade list-style-type) para estilizar listas, a tag < ol> ainda suporta o atributo type com valores como '1', 'a', 'A', 'i', 'I' para definir o tipo de sequência, indicando a ordem é importante.

Como funciona:

  • Propósito: Usada dentro da tag < ol>, ela controla a aparência dos marcadores (os números ou letras que aparecem antes de cada < li>).
  • Valores Comuns (Atributo type):
    • '1' (Padrão): Números arábicos (1, 2, 3).
    • 'a': Letras minúsculas (a, b, c).
    • 'A': Letras maiúsculas (A, B, C).
    • 'i': Algarismos romanos minúsculos (i, ii, iii).
    • 'I': Algarismos romanos maiúsculos (I, II, III).

Exemplo

  1. Primeiro
  2. Segundo
resultaria em: A. Primeiro B. Segundo

Diferença entre type (HTML) e list-style-type (CSS):

  1. O atributo type no HTML é uma forma mais antiga de estilização de listas ordenadas, focando no tipo de sequência.
  2. A propriedade CSS list-style-type oferece um controle muito maior e mais moderno sobre o estilo de listas (ordenadas e não ordenadas), com mais valores como disc, circle, square, decimal-leading-zero, etc..

Quando utilizar:

Use < ol> quando a ordem dos itens for importante (passo a passo, ranking) e < ul> quando a ordem não importa (lista de compras, tópicos). A propriedade type (ou CSS) define como essa ordem é apresentada visualmente.
