Introdução ao HTML

Front-end é a parte de um site ou aplicativo com a qual o usuário interage directamente, responsável pela interface (UI) e experiência do usuário (UX), transformando código em elementos visuais como botões, menus e layouts atraentes, usando principalmente HTML para estrutura, CSS para estilo e JavaScript para interactividade, garantindo que a aplicação seja funcional e agradável em qualquer dispositivo.

Principais Conceitos:

  • O que é: O "lado do cliente", tudo que o usuário vê e toca no navegador ou app.
  • Função: Criar a interface gráfica, garantir usabilidade, acessibilidade e uma experiência fluida.
  • Tecnologias Fundamentais:
    • HTML: Define a estrutura e o conteúdo da página (textos, imagens, links).
    • CSS: Controla o visual (cores, fontes, layout) e responsividade.
    • JavaScript: Adiciona comportamento e interatividade (animações, validação de formulários).
  • Interacção com Back-End: O front-end consome dados de APIs e servidores (back-end) para apresentar informações ao usuário.
  • Objectivo: Proporcionar uma experiência consistente, rápida e acessível, independentemente do dispositivo.

Em resumo, o front-end é a ponte visual e funcional entre o usuário e o sistema, focando na apresentação e interação para criar uma jornada digital agradável e eficiente.
