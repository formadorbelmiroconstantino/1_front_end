Listas em HTML com a Propriedade list-style-type

A propriedade list-style-type em CSS serve para definir o estilo do marcador (ou "bullet") dos itens de uma lista, como círculos (disc), quadrados (square), números (decimal), algarismos romanos (lower-roman), letras (lower-latin), ou até mesmo remover os marcadores (none), personalizando a aparência das listas < ul> e < ol> no HTML.

Exemplos de valores e seus efeitos:

CSS

/* Para listas não ordenadas (< ul>) */

ul { list-style-type: square; /* Quadrados */
list-style-type: disc; /* Círculos (padrão) */
list-style-type: circle; /* Círculos vazios */
list-style-type: none; /* Sem marcador */
}

/* Para listas ordenadas (< ol>) */
ol { list-style-type: decimal; /* 1, 2, 3 (padrão) */
list-style-type: lower-roman; /* i, ii, iii */
list-style-type: upper-alpha; /* A, B, C */
list-style-type: lower-greek; /* α, β, γ */
}

Para que servem na prática?

  • Controlo Visual: Permite que você mude de um marcador simples para algo mais complexo ou temático, como um ícone ou um número específico.
  • Listas Semânticas: Ajuda a diferenciar listas não ordenadas (símbolos) de listas ordenadas (números/letras), mantendo a semântica do documento.
  • Alternativa ao type do HTML: É a forma moderna e recomendada via CSS para estilizar listas, substituindo o antigo atributo type da tag < ol>.
  • Acessibilidade: Usar list-style-type: none; para remover marcadores deve ser feito com cuidado, pois pode impactar leitores de tela; considere alternativas acessíveis se remover os marcadores for essencial.
