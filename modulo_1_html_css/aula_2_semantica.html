Semântica

Semântica HTML é o uso de tags HTML que descrevem o significado e a função do conteúdo, não apenas sua aparência, como < h1 > para título principal, < nav > para navegação, < article > para conteúdo independente, melhorando a acessibilidade (para leitores de tela), SEO (motores de busca) e a manutenção do código, substituindo < div > genéricos por elementos com propósito claro, como < header > , < footer >, < section > e < aside >.

O que é?

Significado e Contexto: Em vez de apenas agrupar elementos visualmente, as tags semânticas (como < header>, < footer>, < nav>, < article>, < section>) comunicam o papel do conteúdo para navegadores, robôs de busca e usuários.
Estrutura vs. Estilo: HTML deve estruturar; CSS deve estilizar. A semântica garante que o código seja lógico, facilitando a aplicação do estilo correcto.

Por que usar?

Acessibilidade: Leitores de tela usam tags semânticas para ajudar usuários com deficiência a navegar. SEO: Motores de busca entendem melhor a estrutura do seu site, o que pode melhorar o ranking. Manutenção: Código mais limpo e organizado é mais fácil de entender e manter por outros desenvolvedores. Interoperabilidade: Facilita a leitura e interpretação por diferentes dispositivos e tecnologias.

Exemplos de tags semânticas
< h1> a < h6>: Títulos e subtítulos.
< p>: Parágrafos.
< a>: Links.
< img>: Imagens.
< nav>: Seções de navegação.
< header>: Cabeçalho do documento ou seção.
< footer>: Rodapé.
< main>: Conteúdo principal da página.
< article>: Conteúdo independente, como um post de blog.
< section>: Agrupa conteúdo temático.
< aside>: Conteúdo complementar (barras laterais).
< ul>, < ol>, < li>: Listas não ordenadas, ordenadas e itens de lista.

Em resumo

Usar semântica HTML é uma boa prática que foca em dar um propósito claro a cada tag, resultando em um site mais robusto, acessível e otimizado, separando estrutura de apresentação.

Se pesquisarmos, por exemplo, por itfocursos no Google e aparecer, significa que os devs do site da ITFocuros, trabalharam na tag head para o SEO por se tratar de uma tag de configuação.
