O que são formulários em HTML?

Formulários HTML são áreas interactivas em páginas web que permitem aos usuários inserir e enviar dados (como nome, email, senhas, mensagens) para um site, usando elementos como campos de texto (< input>), caixas de seleção (< select>), botões (< button>) e checkboxes, fundamentais para logins, cadastros, pesquisas e contacto, sendo a tag < form> a estrutura básica que os envolve e direcciona os dados para um servidor para processamento.

Para que servem

  • Interacção: São a principal forma de o usuário interagir e enviar informações ao site/aplicativo.
  • Colecta de Dados: Coletam informações pessoais para cadastros, pedidos, login e formulários de contato.
  • Comandos: Podem enviar comandos ou solicitações para o servidor, como buscar ou actualizar dados.

Como são feitos/Estrutura e Controlos Principais

  • Tag < form>: Define a área do formulário e seus atributos (como action e method) controlam o envio dos dados.
  • Campos (Widgets): Elementos dentro do formulário para entrada de dados, como:
    • < input type="text"> (campo de texto)
    • < input type="checkbox"> (caixas de selecção múltiplas)
    • < input type="radio"> (botões de rádio para selecção única)
    • < textarea> (para textos longos)
    • < select> (menus suspensos)
    • < button> (para botões de acção, como "Enviar")

Outros Elementos Úteis

  • < textarea>: Para entradas de texto com múltiplas linhas.
  • < select> e < option>: Para criar listas suspensas (dropdowns).
  • < fieldset> e < legend>: Para agrupar campos relacionados e dar-lhes um título.

Exemplo de uso

  • Login: Campos para nome de usuário e senha.
  • Cadastro: Campos para nome, email, CPF, endereço, etc.
  • Contato: Campo de texto para a mensagem e email.
