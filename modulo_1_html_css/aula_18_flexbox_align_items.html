CSS FlexBox | Align Items

Para trabalhar com align-items no HTML/CSS, (Flexbox), você define a propriedade no container pai (com display: flex;) para alinhar os itens filhos no eixo transversal (perpendicular à direcção principal, que por padrão é horizontal) e usar valores como flex-start, flex-end, center, stretch (padrão), ou baseline para controlar seu posicionamento vertical, podendo sobrescrever isso para itens individuais com align-self.

Como usar align-items

1. No HTML: Você tem um container e seus itens filhos.

2. No CSS (Container): Defina o container como flexível e use align-items

.container { display: flex; /* Transforma em container flex */ align-items: center; /* Alinha os itens ao centro no eixo transversal */ height: 200px; /* Para ver o alinhamento vertical */ border: 1px solid black; } .item { padding: 20px; background-color: lightblue; margin: 5px; }

Valores comuns para align-items

  • stretch: (Padrão) Estica os itens para preencher o container no eixo transversal, a menos que tenham altura definida.
  • flex-start: Alinha os itens no início do eixo transversal (topo, se flex-direction for row).
  • flex-end: Alinha os itens no final do eixo transversal (base, se flex-direction for row).
  • center: Centraliza os itens no eixo transversal.
  • baseline: Alinha os itens pela sua linha de base do texto.

Alinhando um item específico: align-self

Para alinhar apenas um item, use align-self no próprio item, com os mesmos valores de align-items:

.item:nth-child(2) { align-self: flex-end; /* O segundo item vai para o final */ }
