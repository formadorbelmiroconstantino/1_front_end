CSS - FlexBox | flex-direction

Trabalhar com Flexbox em HTML envolve transformar um elemento pai (container) com display: flex; para controlar o layout de seus filhos (itens flex), usando propriedades como flex-direction (dircção), justify-content (alinhamento no eixo principal), align-items (alinhamento no eixo transversal), flex-wrap (quebra de linha) e flex-grow/flex-shrink (crescimento/encolhimento dos itens) para criar layouts responsivos e flexíveis, sem precisar de float ou position complexos,.

Conceitos Básicos

Flex Container (Contêiner Flexível): O elemento pai onde você aplica display: flex; ou display: inline-flex;. Todos os seus filhos diretos se tornam "itens flex". Flex Items (Itens Flexíveis): Os filhos diretos do contêiner flexível, cujas propriedades você pode controlar com CSS.

No CSS

.container2{ display: flex; /*Activamos o flexbos e automaticamente, posiciona todos os itens na mesma linha*/ flex-direction: column; /*Elemento em coluna, na vertical*/ }
  • flex-direction: Define a direção dos itens (linha, coluna).
    • row (padrão), row-reverse, column, column-reverse.
  • flex-wrap: Define se os itens quebram para a próxima linha.
    • nowrap (padrão), wrap, wrap-reverse.
  • flex-flow: Atalho para flex-direction e flex-wrap.
  • justify-content: Alinha itens no eixo principal (horizontal, por padrão).
    • flex-start, flex-end, center, space-between, space-around.
  • align-items: Alinha itens no eixo transversal (vertical, por padrão).
    • flex-start, flex-end, center, baseline, stretch.

Propriedades dos Itens (Filhos)

  • flex-grow: Define a capacidade de um item crescer para preencher espaço extra.
  • flex-shrink: Define a capacidade de um item encolher.
  • flex-basis: Define o tamanho inicial do item antes do espaço ser distribuído.
  • align-self: Sobrescreve align-items para um item específico.

Em resumo, abaixo, estão os conceitos para manipular o layout e a aparência em Flexbox:

1. Organizando o Fluxo (o que muitos confundem com "decoration") Se você deseja controlar a direção e o quebra de linha dos itens, utilize o Flex Flow no MDN:

.container { display: flex; flex-flow: row wrap; /* Direção horizontal com quebra de linha */ }

2. Espaçamento entre Itens

Para adicionar "espaço" ou decoração visual entre elementos sem usar margens manuais, utilize a propriedade gap:

.container { display: flex; gap: 20px; /* Cria um espaçamento uniforme entre os itens */ }

3. Alinhamento e Eixos

Para "decorar" o posicionamento dos itens no container:

  • justify-content: Alinha itens no eixo principal (horizontal por padrão).
  • align-items: Alinha itens no eixo perpendicular (vertical por padrão).

4. Decoração de Texto (Text Decoration)

Se você quer decorar o texto dentro de um item flex, use:

.item-flex { text-decoration: underline; /* Sublinhado */ text-decoration-color: blue; /* Cor da linha */ text-decoration-style: dashed; /* Estilo tracejado */ }
