Lógica de programação e algoritmos são os fundamentos para qualquer pessoa que deseja criar softwares ou automatizar tarefas.
1. O que é um Algoritmo?
Um algoritmo é uma sequência finita de passos lógicos e bem definidos para resolver um problema ou realizar uma tarefa. Pense nele como uma "receita de bolo": você segue instruções específicas em uma ordem determinada para chegar ao resultado final. 
2. O que é Lógica de Programação?
É a organização coerente desses passos. É o modo como você estrutura o pensamento para que o computador entenda o que deve ser feito. Enquanto o algoritmo é o "mapa", a lógica é o "raciocínio" por trás da criação desse mapa. 
3. Pilares Fundamentais
Para construir algoritmos, utilizamos conceitos que existem em quase todas as linguagens de programação: 
Variáveis e Tipos de Dados: Espaços na memória para guardar informações (ex: Números, Textos, Booleanos).
Operadores: Matemáticos (+, -, *, /), relacionais (maior que, igual a) e lógicos (E, OU, NÃO).
Estruturas Condicionais: Permitem ao programa tomar decisões (ex: SE a idade for maior que 18, ENTÃO autorize a entrada).
Estruturas de Repetição (Loops): Repetem uma tarefa várias vezes até que uma condição seja atingida (ex: ENQUANTO houver itens na lista, processe o próximo). 

4. Como praticar?

Existem três formas principais de representar e praticar a lógica antes de ir para uma linguagem complexa:
Descrição Narrativa: Escrever o passo a passo em português (ex: "1. Pegue a chave, 2. Insira na fechadura...").
Fluxograma: Representação visual usando figuras geométricas. Você pode usar ferramentas como o Lucidchart para criar os seus.
Pseudocódigo (Portugol): Uma linguagem "intermediária" que usa termos em português para simular o código. O software VisualG é muito utilizado para aprender dessa forma.
